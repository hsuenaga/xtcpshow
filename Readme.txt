This application contains priviledge helper tool.
To uninstall the tool, you need to run xtcpshow-uninstall.sh.
Don't forget use sudo to get root priviledge.

  % sudo xtcpshow-uninstall.sh

This script does:

 (1) unload the helper service from launchd
 (2) delete executable binary of the helper
 (3) delete info.plist of the helper
